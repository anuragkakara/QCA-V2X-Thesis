# Results

This folder contains the empirical data files generated during the experimental evaluation of the QCA framework.

## Contents

Result files will be added as experiments are completed. All data files are in CSV format for ease of analysis and reproducibility.

## Data Files

Empirical results correspond to the experiments reported in Chapter 5 of the thesis:

- AI-NIDS performance metrics (Random Forest vs XGBoost)
- End-to-end latency measurements (Baseline vs QCA-integrated)
- Scalability stress test results across varying batch sizes
- Noise resilience evaluation under simulated corruption

## Dataset Source

All experiments use the publicly available VeReMi dataset (van der Heijden et al., 2018) with the 70/30 stratified train/test split methodology described in Chapter 4.
