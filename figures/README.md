## Design Conventions

Figures in this repository follow a uniform visual style:

- **Vertical panel stacking** for multi-panel figures
- **Linear y-axis** with broken-axis convention where URLLC threshold context is shown
- **Dashed red horizontal line** for the URLLC 10 ms safety threshold reference
- **Blue (#3498db)** for baseline AI-NIDS results
- **Green (#27ae60)** for QCA-integrated results
- **Orange (#e67e22 / #f39c12)** for XGBoost (Figure 9) and F1-Score (Figure B.2)
- **Red (#e74c3c)** for URLLC threshold and recall metric
- Bold value annotations directly on data points
- Summary annotation box at the bottom of each figure

## Format

All figures are saved as PNG files at 300 DPI for print-quality rendering. Each figure is generated by a corresponding Jupyter notebook in the `notebooks/` folder, with the underlying empirical data sourced from `results/empirical_results.json`.

## Reproducibility

To regenerate any figure, open the corresponding notebook in Google Colab and execute the cells. The notebooks use locked empirical values from the experimental results, ensuring perfect consistency with the thesis numbers regardless of the execution environment.
