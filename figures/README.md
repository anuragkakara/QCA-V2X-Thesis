# Figures

This folder contains the figures used in the thesis, generated using matplotlib in Google Colab.

## Contents

Figures will be added as they are generated and finalised. Each figure is exported at high resolution suitable for academic publication.

## Naming Convention

Figures follow the naming pattern: `figure_XX_description.png` where XX corresponds to the figure number in the thesis.

## Format

All figures are saved as PNG files at 300 DPI for print-quality rendering.
